### 2. 数据导入/导出（JSON全量备份）

| 文件 | 改动 |
|------|------|
| **store.js** | 新增 `exportJSON()` —— 导出含学生+任务的完整 JSON 文件，文件名含日期 |
| **store.js** | 新增 `importJSON(file)` —— 解析 JSON 文件，覆盖写入 IndexedDB 和内存状态 |
| **teacher.js (TeacherSettings data)** | 新增 `confirmImport: false` |
| **teacher.js (TeacherSettings methods)** | 新增 `doExportJSON()` / `doImportJSON(e)` |

**注意**：导入操作前有 `confirm()` 确认弹窗，导入后覆盖所有数据。
